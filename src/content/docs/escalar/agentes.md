## Qué es un agente en Claude Code

Un agente es una definición de comportamiento especializado que Claude Code puede invocar como sub-tarea aislada. En vez de pedirle todo a un Claude generalista, delegas tareas específicas a agentes que tienen su propio prompt, sus propias herramientas, y su propio contexto.

Ejemplo: en vez de pedirle a Claude que *"revisa el código y busca bugs de seguridad y además chequea rendimiento"*, lanzas 3 agentes en paralelo — cada uno enfocado en un aspecto — y recibes resultados consolidados.
