## Cuándo necesitas ir más allá del CLI

Claude Code (CLI) es para tu trabajo directo — tú en la terminal, interactuando con Claude. El **Agent SDK** es para cuando quieres que Claude trabaje **dentro de tu aplicación**, sin que el usuario toque la terminal.

| Herramienta | Para quién | Cuándo usarla |
|---|---|---|
| **Claude Code CLI** | Tú trabajando en tu proyecto | Día a día, desarrollo, debugging |
| **Agent SDK (Python)** | Developers que construyen productos | Cuando tu app necesita un agente integrado |
| **Messages API directa** | Developers que quieren control total | Cuando el SDK no cubre tu caso |
