# Atlas Claude en Español

La referencia completa del ecosistema Claude — desde tu primer proyecto hasta producción. En español, con criterio y sin relleno.

## Qué es esto

Obra de referencia pública sobre el ecosistema Claude (Claude Code, MCP, hooks, skills, agentes, API, Managed Agents), destilada desde un hub de conocimiento con más de 100 notas verificadas.

## Estructura

- **Empezar** — Instalar, configurar, primer proyecto
- **Automatizar** — Hooks y skills
- **Conectar** — MCP (Model Context Protocol)
- **Escalar** — Agentes y optimización de modelos/coste
- **Producción** — API, Agent SDK, Managed Agents

## Stack técnico

- [Astro](https://astro.build/) + [Starlight](https://starlight.astro.build/)
- Contenido en Markdown
- Desplegado en Vercel

## Desarrollo local

```bash
npm install
npm run dev
```
